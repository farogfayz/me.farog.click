### Description
**Networking Basics**
- Fundamentals of networking (LANs, WANs, VLANs, WLANs)
- OSI and TCP/IP models
- IP addressing and subnetting (IPv4 and IPv6)
- Ethernet concepts and technologies
- Network topologies and devices (routers, switches, firewalls)

**Cisco Device Basics**
- Cisco IOS (Internetwork Operating System) basics
- Command-Line Interface (CLI) navigation
- Initial configuration of Cisco routers and switches
- Saving and backing up configurations

**Switching and VLANs**
- Basics of switching and MAC addresses
- Configuring VLANs and trunking
- Spanning Tree Protocol (STP) for loop prevention
- Inter-VLAN routing

**Routing Concepts**
- Basics of routing and route tables
- Static routing configuration
- Dynamic routing protocols:
  - RIP (Routing Information Protocol)
  - OSPF (Open Shortest Path First)
- Default routes and administrative distance

**Network Security**
- Access Control Lists (ACLs)
- Securing device access (passwords, SSH)
- Basic firewall concepts
- VLAN security and port security

**Wireless Networking**
- Basics of wireless networks
- WLAN configuration
- Differences between wireless standards (e.g., 802.11a/b/g/n/ac)

**Network Services**
- DHCP (Dynamic Host Configuration Protocol)
- DNS (Domain Name System)
- NAT (Network Address Translation)
- Quality of Service (QoS) basics

**WAN Technologies**
- Basics of WAN technologies (MPLS, VPN, DSL, fiber)
- Configuring basic WAN connections

**Network Troubleshooting**
- Using ping, traceroute, and telnet
- Show and debug commands in Cisco devices
- Troubleshooting IP addressing and connectivity issues
- Identifying and resolving VLAN and routing issues

**Network Automation**
- Introduction to automation tools (e.g., Python, REST APIs)
- Basics of SDN (Software-Defined Networking)
